# Database Web Application Showcase

This repository is a portfolio summary of a team database web application project completed for UBC CPSC 304.  
The original source code was hosted on UBC GitHub and is not publicly shared because it was a team course project.

## Project Overview

This project was a database-driven web application built with a frontend interface, backend routes, and an Oracle SQL database. Users could interact with structured data through web pages and perform common database operations such as searching, filtering, inserting, updating, and deleting records.

## Tech Stack

- Oracle SQL
- Node.js
- JavaScript / TypeScript
- HTML / CSS
- Git

## My Contributions

- Implemented selected frontend pages by following the existing project architecture and coding patterns.
- Developed corresponding backend routes to connect user actions with database operations.
- Wrote SQL queries to support data retrieval, filtering, insertion, update, and deletion features.
- Tested end-to-end functionality between the frontend, backend, and Oracle database.
- Debugged SQL, backend, and integration issues during development.

## Key Features

- Web-based interface for interacting with database records
- SQL-based data retrieval and filtering
- Insert, update, and delete operations
- Backend routes connected to Oracle database queries
- Team-based Git workflow
